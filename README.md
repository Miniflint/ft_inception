## Description
### Presentation
docker and docker-compose goes well together. it's an essential part of any network architecture.
It allows multi docker to communicate to each others using their ports. it also add a layer of security in case of a breach in a docker.

### Goal
Create a docker-compose.yaml that is able to create 3 or 5 (bonus) docker to communicate to each others.
It will create a host database, nginx and mariadb server
